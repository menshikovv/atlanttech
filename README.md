# Twizz Project

Сайт на Next.js с интеграцией Яндекс.Метрики.

## Что уже сделано в коде

Интеграция Метрики уже добавлена в проект:

- Компонент: `components/yandex-metrica.tsx`
- Подключение в layout: `app/layout.tsx`
- Скрипт загружается только если задана переменная `NEXT_PUBLIC_YANDEX_METRIKA_ID`

## Что нужно сделать вам

### 1) Зарегистрировать аккаунт

Нужен обычный аккаунт Яндекса (Yandex ID).  
Если аккаунта нет, создайте его и войдите: https://id.yandex.ru

### 2) Создать счётчик в Яндекс.Метрике

1. Откройте https://metrika.yandex.ru
2. Нажмите **Добавить счётчик**
3. Укажите:
- Имя счётчика (например, `twizz-project`)
- Адрес сайта (например, `https://twizz-project.ru`)
- Часовой пояс (ваш рабочий)
4. Включите нужные опции:
- Вебвизор, карта скроллинга, аналитика форм (обычно включают)
5. Сохраните счётчик

### 3) Взять ID счётчика

После создания получите **числовой ID** счётчика (пример: `12345678`).

## Настройка проекта

### 4) Добавить переменную окружения

Откройте файл `.env` в корне проекта и добавьте строку:

```env
NEXT_PUBLIC_YANDEX_METRIKA_ID=12345678
```

Где `12345678` замените на ваш реальный ID счётчика.

### 5) Перезапустить приложение

После изменения `.env` перезапустите dev-сервер:

```bash
npm run dev
```

## Проверка, что Метрика работает

1. Откройте сайт в браузере
2. Перейдите в Яндекс.Метрику -> ваш счётчик -> Отчёты/Посещаемость
3. Убедитесь, что визиты появляются
4. Для быстрой проверки откройте DevTools -> Network и убедитесь, что есть запросы к `mc.yandex.ru`

## Как отправлять цели (события) вручную

Если нужно трекать клики/отправку формы, можно вызывать `ym`:

```ts
if (typeof window !== 'undefined' && typeof window.ym === 'function') {
  window.ym(12345678, 'reachGoal', 'FORM_SUBMIT')
}
```

Рекомендуется вынести ID в переменную, чтобы не хардкодить его в коде.

## Важные примечания

- Если `NEXT_PUBLIC_YANDEX_METRIKA_ID` не задан, скрипт не подключается.
- На localhost данные в отчётах могут появляться с задержкой.
- Для продакшена проверьте, что домен в настройках счётчика совпадает с реальным доменом сайта.
